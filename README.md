# Portfólio Profissional

Portfólio pessoal desenvolvido para centralizar projetos, estudos e experiências em desenvolvimento de software, com foco em organização, escalabilidade e automação de conteúdo.

## 🎯 Objetivo

Este projeto tem como objetivo servir como um ponto central de apresentação profissional, permitindo a exibição dinâmica de projetos, habilidades técnicas e certificações, sem a necessidade de alterações frequentes no código-fonte.

## 🧠 Conceito

O portfólio foi projetado para ser:

* **Escalável** — acompanhando a evolução profissional ao longo do tempo
* **Automatizado** — carregando dados diretamente de um banco de dados
* **Manutenível** — com separação clara entre dados, lógica e apresentação
* **Dinâmico** — novos conteúdos podem ser adicionados diretamente no banco

## 🧱 Estrutura do Projeto

O portfólio é composto pelas seguintes seções principais:

* **Home** — apresentação e projetos em destaque
* **About** — trajetória e áreas de interesse
* **Projects** — lista completa de projetos desenvolvidos
* **Certificates** — certificados obtidos ao longo da formação
* **Currículo** — acesso direto ao currículo em PDF

## 🗄️ Arquitetura de Dados

Os conteúdos exibidos no portfólio são carregados dinamicamente a partir de um banco de dados.

Estrutura principal:

* `skills` — tecnologias e ferramentas utilizadas
* `projects` — projetos desenvolvidos
* `certificates` — certificados e formações

Arquivos estáticos como imagens e documentos são armazenados no **Storage**.
